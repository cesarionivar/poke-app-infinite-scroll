# Poke App

This is an app using the API of PokeAPI

```
https://pokeapi.co/
```

## Technologies
- Html
- CSS
- Js
- Sass
- Webpack

## To run it
Build the dependencies 
```
npm install
```

then 

```
npm run build
```

or to run it as developmet 
```
npm run dev
```
